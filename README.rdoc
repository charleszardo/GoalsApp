== Goals App

Simple rails app that allows users to list goals and track completion. Goals may be listed as public or private.  Users can comment on goals or other users.  Users can also cheers goals.  App features unit, service, and integration testing using RSpec, Capybara, and Factory Girl.

=== To Run

* Install dependencies: <tt>bundle install</tt>

* Create database: <tt>rake db:create</tt>

* Start server: <tt>rails s</tt>
